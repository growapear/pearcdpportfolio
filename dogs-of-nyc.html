Dogs of NYC

A playful scrollytelling exploration of New York City’s most popular dog names—by borough, gender, and neighborhood.

Visit Project Website →

Overview

Dogs of NYC is a visual storytelling project that transforms a seemingly simple dataset—registered dog names in New York City—into a poetic portrait of place, personality, and naming culture. Using data from the NYC Open Data portal, the project presents the most common dog names across boroughs, zip codes, and gender through an interactive scrollytelling website. But beyond just presenting facts, the project plays with color, motion, and typographic rhythm to create a joyful and intuitive experience. As users scroll, they uncover subtle trends—like “Max” dominating in Manhattan, or more playful names like “Peanut” clustering in Brooklyn. The design doesn’t aim to be analytical alone, but emotional: it asks how names carry identity, and how urban data can reflect the personalities of both dogs and neighborhoods. The interface embraces simplicity, using bright palettes and large type to create moments of pause and delight. Each transition reveals not just numbers, but the lived texture of urban life.

Process

The project began with data sourced from the NYC Open Data Portal, specifically the dog licensing dataset. I explored the raw CSV to identify naming patterns across boroughs, zip codes, gender, and frequency. Data cleaning was done using Excel and Python, followed by exploratory charting in Observable to test visual directions. Once key trends were identified, I shifted to designing a scrollytelling narrative—building a lightweight, custom website using HTML, CSS, and JavaScript. Throughout the process, I focused on graphic clarity, color hierarchy, and pacing between data and text. The final design embraces playfulness without sacrificing legibility—transforming a public dataset into a spatial, emotional reading of city life through its dogs.

Technical Details

The system utilizes:

Data Processing

Python and pandas for cleaning and analyzing NYC Open Data's dog licensing dataset, processing over 500,000 records

Geospatial Analysis

Integration with NYC's PLUTO dataset for neighborhood-level analysis and mapping of dog population patterns

Interactive Visualization

D3.js and Mapbox GL JS for creating responsive, web-based visualizations and interactive maps

Statistical Analysis

Custom algorithms for identifying trends in breed distribution, name popularity, and neighborhood correlations
