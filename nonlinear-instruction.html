Nonlinear Instruction

A speculative learning interface that translates craft gestures into a nonlinear system of instruction, memory, and motion.

Visit Project Website →

Overview

Nonlinear Instruction is a research and interaction design project exploring how embodied craft knowledge might be preserved and taught through digital systems—without relying on linear, text-based formats. Traditional craft learning is passed down through observation, repetition, and intuition. Unlike institutional education, it is dynamic and nonlinear. This project asks: What if a system could teach like an artisan—through motion, ambiguity, and experience? The prototype is an interface inspired by pottery instruction. Instead of presenting steps in order, it invites users to explore floating gesture tiles. Each tile represents a hand movement and can be grouped, rearranged, or followed based on curiosity and context. Users can toggle between timeline views, gesture types, and cultural interpretations. At the heart of the project is the belief that teaching doesn't have to be fixed—it can be responsive, spatial, and intuitive. This work directly informed the design logic behind Crafticulate, offering an early framework for non-verbal, motion-based interfaces that preserve embodied knowledge without reducing it to instruction manuals.

Process

The project began as an exploration into how gesture and memory could be encoded digitally. I began by analyzing how artisans teach—through proximity, repetition, and subtle correction—and asked how that could be reimagined through a digital interface. Inspired by floating spatial interfaces, I prototyped a UI system where gestures function as interactive tiles. Each tile is data-rich: it links to a specific process (e.g. pulling, pinching), a material context, and a sequence or loop within the making process. The layout is intentionally non-linear, offering multiple paths for interpretation, experimentation, and memory construction.

Technical Details

The system utilizes:

Hand Tracking via Webcam

A live camera feed detects and tracks hand positions in real time using computer vision libraries

Gesture-Driven Interaction with Projector Overlay

A spatial projection overlays digital tiles onto physical surfaces, allowing users to interact with the system using hand movement alone

UI Prototyping in p5.js

The interface was developed using p5.js, enabling responsive gesture interaction, animated transitions, and flexible experimentation with motion-based navigation

Framework Developed with Spatial Pixel's Procession Tool

The core interaction engine leverages Procession, a spatial computing framework by Spatial Pixel, to create intuitive, embodied UI behaviors based on user movement
